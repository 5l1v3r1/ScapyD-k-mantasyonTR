* TCP

```python
tcp = TCP()

tcp.display()
tcp.dport = 80
```

* ICMP

```python
icmp = ICMP()
icmp.show()
icmp.display()
icmp.type = "echo-request"
```

* IP

```python

```

* ARP

```python
arp = ARP()

arp.display()

```

